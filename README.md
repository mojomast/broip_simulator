## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/broipsimulator.git
   cd broipsimulator
   ```

2. Install dependencies:
   ```
   cd broip-cli
   npm install
   ```

3. Build the application:
   ```
   npm run build
   ```

4. Run the application:
   ```
   npm start
   ```
